# 🎮 Jogo - Palavra da Vez

Um mini jogo desenvolvido com **HTML, CSS e JavaScript** onde o jogador precisa descobrir as palavras corretas através das dicas exibidas na tela.

---

## ✨ Sobre o projeto

O projeto funciona como um jogo simples de adivinhação:

- O usuário digita uma palavra no input
- O sistema verifica se ela corresponde a uma das palavras secretas
- Caso esteja correta, a palavra é revelada na tela
- As dicas ajudam o jogador a descobrir cada resposta

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
